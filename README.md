# noomi-relaen
[noomi](https://www.npmjs.com/package/noomi) [relaen](https://www.npmjs.com/package/relaen)支撑包，用于noomi和relaen的结合使用。

## 安装
npm i noomi-relaen -S

## 编译

1. 需安装 `typescript`,`ts-node`。  
2. 删除 `dist` 目录（如果有）。
3. 在当前目录执行命令 `tsc`。

## 其他说明

需要配合 [relaen](https://www.npmjs.com/package/relaen) 包使用。
